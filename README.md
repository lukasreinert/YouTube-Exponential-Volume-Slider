# YouTube Exponential Volume Slider
Makes the YouTube video volume slider exponential so it's easier to select lower volumes.

Humans perceive loudness logarithmically, so exponential scaling makes the slider feel more natural and precise at low volumes.

## Installation

### Recommended Method

Visit the script page on Greasy Fork and click **Install**:  
[https://greasyfork.org/en/scripts/555739](https://greasyfork.org/en/scripts/555739)

### Manual Installation

1. Open your user script manager (e.g., Tampermonkey).  
2. Create a new user script.  
3. Copy and paste the code from [`youtube_exponential_volume_slider.js`](https://raw.githubusercontent.com/lukasreinert/YouTube-Exponential-Volume-Slider/refs/heads/main/youtube_exponential_volume_slider.js).  
4. Save the script and reload YouTube.
